# 🏆 FuelFit | Premium Luxury Gym Platform

**The ultimate high-conversion landing page and digital front desk for high-end fitness centers.**

FuelFit is a production-ready, ultra-premium web application built for gym owners who want to automate their lead generation and professionalize their brand. It transforms a traditional gym's digital presence from a simple informational page into a **24/7 Sales Assistant**.

---

## 🚀 Key Features

### 1. **Smart BMI & Macro Calculator**
A highly interactive tool that engages visitors by providing real-time fitness metrics. 
- **Lead Generation**: Recommends specific membership plans based on user goals.
- **Value First**: Provides instant BMI and Macro targets to build trust before asking for a sale.

### 2. **Interactive Before/After Slider**
A cinematic visual comparison tool that allows prospects to see the results of your gym's training programs in real-time.
- **Social Proof**: High-impact results visualization.
- **Micro-interactions**: Smooth dragging and grayscale-to-color transitions.

### 3. **Lead Capture Pipeline**
Multiple high-conversion touchpoints designed to turn visitors into walk-ins.
- **Floating "Free Day Pass" Button**: Persistent CTA that appears on scroll to capture interest at its peak.
- **Join Modals**: Seamless, friction-free forms for membership inquiries.

### 4. **Premium "Obsidian" Design System**
A bespoke, dark-luxury aesthetic designed to position the gym as a premium choice.
- **Modern Stack**: Built with Next.js 15, Tailwind CSS, and Framer Motion for buttery-smooth animations.
- **Mobile-First**: Fully responsive experience that feels like a native app on all devices.

### 5. **Local Business Authority**
Optimized for local SEO and logistical clarity.
- **Dynamic Hours**: Real-time "Open Now" status.
- **SEO Ready**: Enhanced metadata and OpenGraph tags for maximum search visibility.

---

## 🛠️ Tech Stack

- **Framework**: [Next.js 15 (App Router)](https://nextjs.org/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Fonts**: Inter & Oswald (via Google Fonts)

## 🏁 Getting Started

### Prerequisites
- Node.js 18.x or later
- npm / yarn / pnpm

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/your-repo/gym-demo.git
   cd gym-demo
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```
